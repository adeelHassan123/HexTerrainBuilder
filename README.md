# HexTerrainBuilder

A 3D hexagonal terrain builder application built with React, Three.js, and TypeScript. Create, edit, and manage hexagonal grids in an interactive 3D environment.

## Tech Stack

- **React 19** - UI framework
- **TypeScript** - Type safety
- **Vite** - Build tool with HMR
- **Three.js** - 3D rendering
- **@react-three/fiber** - React renderer for Three.js
- **@react-three/drei** - Useful Three.js abstractions
- **Zustand** - State management with persistence
- **Zundo** - Temporal/undo-redo for Zustand
- **Tailwind CSS** - Utility-first styling

## Installation

```bash
npm install
# or
pnpm install
# or
yarn install
```

## Development

Start the development server:

```bash
npm run dev
```

## Building

Create an optimized production build:

```bash
npm run build
```

Preview the production build:

```bash
npm run preview
```
